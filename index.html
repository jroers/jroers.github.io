PROJECTS

SangRouge

SangRouge is a blood collection system - and my first healthcare-related web app! When it comes to the donation process, donors that give blood at multiple organizations often need to manage multiple accounts and fill out duplicative forms. With SangRouge, a donor's profile follows them from organization to organization. This ensures that the donors wait the appropriate amount of time between donations, and that donors only have one account to manage.

FEATURES: Multiple user types, Conditional form validations, Updating navbar logo with the logged-in tech's organization name/logo

TECHNOLOGIES: Ruby on Rails, Bcrypt, PostgreSQL

Kon-ju

As a private French tutor, I have found that my students often struggle with information overload using translation/conjugation websites. Kon-ju aims to help students without overloading them. By only displaying a single tense of verb, the user is able to hone in on only the conjugations that matter to them.

Should a verb not exist in the database, the site will look at the structure of the infinitive and provide regular conjugations so a user doesn't need to recreate the wheel.

FEATURES: Built-in regular conjugation logic, List creator

TECHNOLOGIES: Node.js, Express.js, MongoDB
